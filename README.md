
# Giày Da Nam Đà Nẵng - E-commerce Demo

Demo website bán giày da nam được dựng bằng React, Vite và Tailwind CSS. Bản này phù hợp để deploy lên Vercel cho khách hàng xem giao diện, luồng sản phẩm, giỏ hàng và checkout mẫu.

## Chạy local

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

Thư mục build sẽ nằm trong `dist`.

## Deploy Vercel

- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

Lưu ý: đây là bản demo frontend. Checkout, tồn kho, thanh toán và quản trị sản phẩm đang là dữ liệu mẫu, chưa kết nối backend.
